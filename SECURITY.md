# Security Policy

## Supported Versions

All versions of EggyHax.

| Version | Supported          |
| ------- | ------------------ |
| 4.0.0x - 4.0.1x   | :white_check_mark: |
| 3.0.0x - 3.9.9x   | :white_check_mark:  |
| 2.5.1 - 2.9.9x   | :white_check_mark: |
| < 2.5.0  | :x:                |

Use this section to tell people how to report a vulnerability.
